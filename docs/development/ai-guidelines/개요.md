# 📖 개요

이 문서는 LLM에게 **Skill 기반 능력**(코드 생성, 분석, 변환, 자동화 등)과 **MCP 기반 도구 체계**(Web, File, Database, Workflow 등)을 최대한 활용하게 만들기 위한 템플릿입니다.

**용도:**
- Claude Code에 복사하여 사용
- 프로젝트별 커스터마이징
- AI 페어 프로그래밍 세션 시작 시 컨텍스트 제공
