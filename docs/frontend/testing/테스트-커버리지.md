# 테스트 커버리지

## 커버리지 설정 (vitest.config.ts)

```typescript
test: {
  coverage: {
    provider: 'c8',
    reporter: ['text', 'json', 'html', 'lcov'],
    
    // 제외 대상
    exclude: [
      'node_modules/',
      'tests/',
      '**/*.d.ts',
      '**/*.config.*',
      '**/mockData',
      'dist/',
      '.nuxt/',
      '.output/',
      'pages/**',  // E2E로 커버
      'app/layouts/**',  // E2E로 커버
      'app/providers/**'  // 설정 파일
    ],
    
    // 커버리지 목표
    statements: 80,
    branches: 75,
    functions: 80,
    lines: 80,
    
    // 경고로 처리 (빌드 실패 방지)
    all: true,
    skipFull: false
  }
}
```

## 커버리지 리포트 생성

```bash
# 커버리지 생성
npm run test:coverage

# HTML 리포트 확인
open coverage/index.html
```
